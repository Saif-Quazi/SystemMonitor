# System Monitor

A lightweight, always-on-top system monitoring widget for Windows, displaying CPU, GPU, and RAM usage in real-time.

**Dependencies:** psutil, GPUtil, pystray, Pillow, PySide6

## Installation
Download the latest release from the [GitHub Releases](https://github.com/Saif-Quazi/SystemMonitor/releases) page and run the executable.

## Usage
The widget displays CPU, GPU, and RAM usage. Access settings via the system tray icon.

## Local Setup
For development or local use:
1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `python main.py`
